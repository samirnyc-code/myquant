## Was ich heute gelernt habe

### Konzepte

| Begriff | Was es ist |
|--------|------------|
| **VS Code** | Code-Editor (kostenlos, von Microsoft) |
| **Git** | Lokales Tool für Versionskontrolle — trackt jede Änderung am Code |
| **GitHub** | Cloud-Hosting für Git-Repos — dein Code ist online gesichert |
| **Repository (Repo)** | Ein Ordner/Projekt das von Git verwaltet wird |
| **Clone** | Eine Kopie eines GitHub-Repos lokal auf deinen Computer laden |
| **Commit** | Eine gespeicherte Änderung im Git-System |
| **Command Palette** | Cmd+Shift+P — Suchleiste für alle VS Code Befehle (P = Palette) |
| **Extension** | Plugin das VS Code erweitert |
| **GitLens** | VS Code Extension — zeigt wer was wann geändert hat |
| **Claude Code** | Anthropic's AI direkt in VS Code integriert |
| **GitHub Copilot** | Microsoft's AI — NICHT dasselbe wie Claude Code, nicht installiert |
| **Settings Sync** | VS Code Einstellungen werden über Geräte hinweg synchronisiert |
