## Summary
A downloadable NinjaTrader 8 chart template ("VeritasOrderFlow") post, not an educational article. The template combines three MZpack indicators — mzFootprint, mzVolumeProfile, and mzMarketDepth — into one pre-configured layout. Installation is by copying the XML file to Documents\NinjaTrader 8\templates\. Requires MZpack 3.16.3 or later. The article directs readers to the User Guide and Video Guide for indicator details.

## Key concepts & definitions
- VeritasOrderFlow template: pre-configured NT8 chart layout bundling mzFootprint (footprint), mzVolumeProfile (volume profile), and mzMarketDepth (DOM/liquidity heatmap).
