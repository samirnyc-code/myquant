## Summary
Brief annotated post (author: Konstantin/KO) showing two intraday NQ 03-19 long trades on M5 + M1 charts using the "NASDAQ Workspace 1.1" template with mzBigTrade, mzMarketDepth, mzVolumeProfile and mzVolumeDelta. Trade 1 (long) was closed near the weekly VAL and VWAP. Trade 2 (long) was opened near the same VAL with confirmation from increasing Delta and ATR on M5 and several aggressive BUY trades supporting the uptrend; it also closed at similar levels.

## Key concepts & definitions
- Weekly VAL used as the key location — both entry (trade 2) and exit (trade 1) anchored near VAL and VWAP.
- Confirmation for trade 2: Delta and ATR increasing on M5, price stopped near VAL, then several aggressive BUY trades read as uptrend confirmation.
- Standardized chart environment via a shared workspace template (NASDAQ Workspace 1.1).
