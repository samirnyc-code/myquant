## Base signal

Signal price + StopPrice from the indicator. This spec is a filter/execution layer only.

## Filters (ALL must pass; all causal at signal-bar close)
1. **No counter-IB-break.** IB = first 12×5M bars (8:30–9:30 CT). Break state from the
   CLOSE of the first post-IB bar breaking IB-high (up) / IB-low (down). Skip Long while
   state = down-only; skip Short while up-only. Pre-break / aligned / both-broken: keep.
2. **No prior trend day.** Skip the whole session if prior session range > 1.6 × ADR14
   (14-day rolling mean of daily RTH ranges, as of the prior day).
3. **Before 13:00 CT.** Signal bar must close before 13:00.

## Execution
- Entry: market at next bar open (+1 tick slip modeled).
- Stop: indicator StopPrice, 1 tick beyond. R = |fill − stop| (median ~16 pts, varies).
- Target: 3R limit. NO break-even ratchet, NO trail (BE tested worse on this book).
- Flat at session close 15:14 CT (EOD exit ≈ 49% of trades; carries the bulk of profit).
- No-hedge, first-in-wins: skip any signal opposite to an open position (~5% skipped;
  −0.008R measured cost). Same-direction stacking allowed (max observed 10).
- 1 MES per signal; $2.00 RT.
