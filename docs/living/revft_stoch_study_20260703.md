# Methodology notes

- **In-sample:** all analysis uses the same 5yr set — findings are hypotheses, not out-of-sample results.
- **PeriodD=1:** D ≡ K; K/D-cross features are degenerate. Re-export stoch with PeriodD=3 for a real signal line.
- **Minimum threshold for a 'finding':** R 95%-interval must exclude zero AND hold year-by-year.
- **STO_K_next** is shown as a ceiling for what a future-informed K-slope could achieve.
- **Roll sensitivity:** ~PeriodK bars (8 bars = 40 min) around each roll may have a spurious K spike due to price discontinuity from Panama back-adjustment. Estimated <1% of all bars affected.
