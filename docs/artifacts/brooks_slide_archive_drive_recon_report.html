It is the missing "concept layer" between our 4 mined books and the 2,049 real trading days from the forum.

Inside every topic folder

A rigid, machine-friendly scheme: 00.png is the title/intro slide, 01.png…NN.png are annotated course slides (real ES 5-min charts, Brooks' drawings + bullet text), and M-01.png… are companion variants that appear to be the bare, un-annotated charts. Each slide's footer cites its source, e.g. Part 5 (T-Z) Slide 546. Google Drive OCR already returns clean slide text via the API, so the teaching text is programmatically extractable.

How this plugs into the Codex

  1. The ontology becomes the Codex spine. ~456 folder names are a ready-made pattern/variant taxonomy — crucially including failure variants as first-class siblings ("Wedge Bottom" / "Wedge Bottom but Bear BO" / "Wedge Bottom Failed Bear BO"). Map our 91 canonical setups onto it and every gap in our setup list becomes visible immediately.
  2. A Slide Explorer next to the Figure Explorer. Same UI we already built for the 459 book figures: OCR each slide, index by topic, full-text search, zoom. Books teach in prose; these teach in annotated examples — different artifact, same shell.
  3. The M-files are a built-in trainer. Un-annotated chart + annotated twin = a natural "predict, then reveal" drill: show the bare chart, you mark the pattern and entry, flip to Al's annotated version. This is the Replay Trainer idea with zero chart-generation work.
  4. Cross-link concept ↔ reality. Forum bar-by-bar days (now 2,049 of them, not 949 — the old scraper silently truncated) are tagged with Brooks shorthand (H2, W, DT…). Each tag can deep-link to its slide-topic folder: read the concept, then see it on ten real trading days, then step it bar by bar.
  5. Base-rate claims to test. Slides state probabilities ("80% chance of at least one reversal in first hour after a big gap down"). With 2,049 labeled real days we can measure Al's own claimed base rates against his own charts — honest numbers, no backtest hopium.
Provenance flag: this is paid Brooks Trading Course material sitting on a third party's Drive. Fine as personal study input, but it should not be redistributed — keep any Codex integration in your private, gitignored/Drive-local copy, same as the book PDFs.

Overlap with what we already have

No overlap with the forum bar-by-bar threads or the 1,221 blog EOD charts. Conceptual overlap with the 4 books (same patterns) but entirely distinct artifacts — the slides are example-driven, the books prose-driven. If you own the course videos, these are their still frames, reorganized by topic; that makes them the best "lookup by pattern name" asset we have.

Caveats

Every folder was enumerated by name; per-leaf file counts are extrapolated from six sampled leaves (uniform structure, 3–28 files each). Three slides were content-read to verify what they are. The two Dec-2025 webp files couldn't be OCR'd via the API. The parent folder above Slides exists but wasn't accessible from the share link — worth asking the owner what else is up there.
